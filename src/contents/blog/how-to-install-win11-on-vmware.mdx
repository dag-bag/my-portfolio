## Step 1 - Download Windows 11 ISO

  <AlertTitle>Link</AlertTitle>
  <AlertDescription>[Download Windows 11 ISO](https://www.microsoft.com/en-us/software-download/windows11)</AlertDescription>
</Alert>

1. Select Windows 11 (multi-edition ISO)
2. Click Download
3. Select your language
4. Click Confirm

## Step 2 - Create Windows 11 Virtual Machine

1. Click Create a New Virtual Machine

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-2(1).png'
  alt='Create a New Virtual Machine'
  width={1532}
  height={717}
/>

2. Select Typical
3. Next

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-2(2-3).png'
  alt='Select Typical'
  width={428}
  height={430}
/>

4. Select Installer disc image file (iso)
5. Click Browse to select the ISO that your download before
6. Next

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-2(4-6).png'
  alt='Select ISO and click next'
  width={428}
  height={430}
/>

7. Select Microsoft Windows
8. Select Windows 10 x64 or Windows 10 and later x64 (If it has this option)
9. Next

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-2(7-9).png'
  alt='Select operating system and click next'
  width={428}
  height={430}
/>

10. Type the Virtual machine name, here I named it Windows 11
11. Click Browse to select the virtual machine install location, here I select E drive
12. Next

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-2(10-12).png'
  alt='Name your virtual machine and click next'
  width={428}
  height={430}
/>

13. Type the Maximum disk size, recommended 60GB or more
14. Select Store virtual disk as a single file
15. Next

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-2(13-15).png'
  alt='Set disk size and click next'
  width={428}
  height={430}
/>

16. Finish

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-2(16).png'
  alt='Finish'
  width={428}
  height={430}
/>

## Step 4 - Install Windows 11

1. Select your preferred language
2. Next

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-4(1-2).png'
  alt='Select your preferred language'
  width={1024}
  height={768}
/>

3. Click Install now

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-4(3).png'
  alt='Click Install now'
  width={1024}
  height={768}
/>

4. Select I don't have a product key or you may type your key

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-4(4).png'
  alt='Type your key or continue without a key'
  width={1024}
  height={768}

5. Select the version that you want to install
6. Next

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-4(5-6).png'
  alt='Select the version that you want to install'
  width={1024}
  height={768}
/>

<Alert icon='💡' variant='warning'>
  <AlertTitle>Tips</AlertTitle>
  <AlertDescription>
    If the below situation does not appear, please skip to step 19.

    Press <Kbd>Shift</Kbd> + <Kbd>F10</Kbd> to open CMD
  </AlertDescription>
</Alert>

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/cant-install.png'
  alt='This PC cannot run Windows 11'
  width={1024}
  height={768}
/>

7. Type `regedit`

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-4(7).png'
  alt='Type regedit'
  width={1024}
  height={768}
/>

8. Go to `Computer\HKEY_LOCAL_MACHINE\SYSTEM\Setup`
9. Right click Setup
10. New > Key, and name it `LabConfig`

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-4(8-10).png'
  alt='Create a new key'
  width={1024}
  height={768}
/>

11. New > DWORD (32-bit) Value

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-4(11).png'
  alt='Create a new DWORD (32-bit) Value'
  width={1024}
  height={768}
/>

12. Name it `BypassTPMCheck`
13. Value is 1
14. OK
15. Close the regedit (Registry Editor)

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-4(12-15).png'
  alt='Set the value to 1'
  width={1024}
  height={768}
/>

16. Close the Windows Setup
17. Yes

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-4(16-17).png'
  alt='Close the Windows Setup'
  width={1024}
  height={768}
/>

18. Install now

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-4(18).png'
  alt='Install now'
  width={1024}
  height={768}
/>

19. Accept the Terms
20. Next

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-4(19-20).png'
  alt='Accept the Terms'
  width={1024}
  height={768}
/>

<Image
  src='/static/images/blog/how-to-install-win11-on-vmware/step-4(21).png'

  src='/static/images/blog/how-to-install-win11-on-vmware/step-4(22-23).png'
